The `Navigation` component provides the main navigation menu for the portfolio website. It includes route links with active state highlighting and a theme toggle button.

## Features

- Responsive layout (column on mobile, row on desktop)
- Active route highlighting with underline decoration
- Theme toggle with icon and label
- Semantic HTML with proper ARIA labels
- Built with Next.js Link for client-side routing

## Theme toggle

The navigation includes a theme toggle button that:

1. Shows the current theme icon (Moon for dark, Sun for light)
2. Displays the current theme label ("Dark" or "Light")
3. Toggles between light and dark modes on click
4. Handles system theme detection

<CodeGroup>
```tsx Theme toggle behavior
const toggleTheme = () => {
  if (theme === "system") {
    setTheme(resolvedTheme === "dark" ? "light" : "dark");
  } else {
    setTheme(theme === "dark" ? "light" : "dark");
  }
};
```

```tsx Theme icon rendering
const getThemeIcon = () => {
  if (theme === "system") {
    return resolvedTheme === "dark" ? (
      <Moon className="w-4 h-4" />
    ) : (
      <Sun className="w-4 h-4" />
    );
  }
  return theme === "dark" ? (
    <Moon className="w-4 h-4" />
  ) : (
    <Sun className="w-4 h-4" />
  );
};
```
</CodeGroup>

## Active route detection

The component highlights the active route using pathname comparison:

```typescript
const isActive = to === "/" 
  ? pathname === "/" 
  : pathname.startsWith(to);
```

- **Home route** (`/`) - Only active when pathname is exactly `/`
- **Other routes** - Active when pathname starts with the route path

Active links receive an `underline` class for visual indication.

## Accessibility

The navigation implements several accessibility features:

```tsx
<nav aria-label="Main navigation">
  <ul className="flex md:gap-8 gap-2 md:text-lg font-medium">
    {routes.map(({ to, label }) => (
      <li key={to}>
        <Link href={to} className={cn(/* ... */)}>
          {label}
        </Link>
      </li>
    ))}
  </ul>
  <button
    onClick={toggleTheme}
    aria-label={`Switch to ${resolvedTheme === "dark" ? "light" : "dark"} mode`}
  >
    {getThemeIcon()}
    <span>{getThemeLabel()}</span>
  </button>
</nav>
```

- Semantic `nav` element with `aria-label`
- List structure for navigation items
- Descriptive `aria-label` on theme toggle button
