The portfolio website includes three core UI components built with Radix UI primitives and styled with TailwindCSS. These components provide consistent styling and behavior across the application.

## Button

A versatile button component with multiple variants and sizes, built with Radix UI Slot and class-variance-authority.

### Usage

<CodeGroup>
```tsx Default button
import { Button } from "@/components/ui/button";

export default function Page() {
  return (
    <Button onClick={() => console.log("Clicked!")}>
      Click me
    </Button>
  );
}
```

```tsx Button variants
import { Button } from "@/components/ui/button";

export default function Page() {
  return (
    <div className="flex gap-2">
      <Button variant="default">Default</Button>
      <Button variant="destructive">Delete</Button>
      <Button variant="outline">Outline</Button>
      <Button variant="secondary">Secondary</Button>
      <Button variant="ghost">Ghost</Button>
      <Button variant="link">Link</Button>
    </div>
  );
}
```

```tsx Button sizes
import { Button } from "@/components/ui/button";

export default function Page() {
  return (
    <div className="flex items-center gap-2">
      <Button size="sm">Small</Button>
      <Button size="default">Default</Button>
      <Button size="lg">Large</Button>
    </div>
  );
}
```

```tsx Icon buttons
import { Button } from "@/components/ui/button";
import { X } from "lucide-react";

export default function Page() {
  return (
    <div className="flex gap-2">
      <Button size="icon">
        <X />
      </Button>
      <Button size="icon-sm">
        <X />
      </Button>
      <Button size="icon-lg">
        <X />
      </Button>
    </div>
  );
}
```

```tsx As child (polymorphic)
import { Button } from "@/components/ui/button";
import Link from "next/link";

export default function Page() {
  return (
    <Button asChild>
      <Link href="/contact">Contact Us</Link>
    </Button>
  );
}
```
</CodeGroup>

### Props

<ParamField path="variant" type="string" default="default">
  Button style variant
  
  Options:
  - `default` - Primary button with brand background
  - `destructive` - Red button for destructive actions
  - `outline` - Button with border and transparent background
  - `secondary` - Secondary button style
  - `ghost` - Transparent button that shows background on hover
  - `link` - Text-only button styled as a link
</ParamField>

<ParamField path="size" type="string" default="default">
  Button size variant
  
  Options:
  - `sm` - Small button (h-8, rounded-md, gap-1.5, px-3)
  - `default` - Default button (h-9, px-4, py-2)
  - `lg` - Large button (h-10, rounded-md, px-6)
  - `icon` - Icon-only button (size-9)
  - `icon-sm` - Small icon button (size-8)
  - `icon-lg` - Large icon button (size-10)
</ParamField>

<ParamField path="asChild" type="boolean" default={false}>
  When `true`, Button renders as a Radix UI Slot, allowing you to compose it with other components like Next.js Link. The Button styles will be applied to the child element.
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply to the button
</ParamField>

<ParamField path="...props" type="React.ComponentProps<'button'>">
  All standard HTML button attributes (onClick, disabled, type, etc.)
</ParamField>

### Features

- **Focus states** - Ring indicator on keyboard focus
- **Disabled states** - Automatic opacity and pointer events handling
- **SVG sizing** - Automatic icon sizing with `[&_svg]:size-4`
- **Accessibility** - Supports `aria-invalid` for form validation
- **Dark mode** - Theme-aware colors and states

## Badge

A small label component for tags, status indicators, and metadata. Built with Radix UI Slot and class-variance-authority.

### Usage

<CodeGroup>
```tsx Basic badge
import { Badge } from "@/components/ui/badge";

  return <Badge>New</Badge>;
}
```

```tsx Badge variants

  return (
    <div className="flex gap-2">
      <Badge variant="default">Default</Badge>

      <Badge variant="destructive">Error</Badge>
      <Badge variant="outline">Outline</Badge>
    </div>
  );
}
```

```tsx With icons
export default function Page() {
  return (
    <div className="flex gap-2">

```

```tsx As child (link badge)

    <Badge asChild variant="outline">

```
</CodeGroup>

### Props

<ParamField path="variant" type="string" default="default">
  Badge style variant
  
  Options:
  - `default` - Primary badge with brand background
  - `secondary` - Secondary badge style
  - `destructive` - Red badge for errors or warnings
  - `outline` - Badge with border and transparent background
</ParamField>

<ParamField path="asChild" type="boolean" default={false}>
  When `true`, Badge renders as a Radix UI Slot, allowing composition with other components like links
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply to the badge
</ParamField>

<ParamField path="...props" type="React.ComponentProps<'span'>">
  All standard HTML span attributes
</ParamField>

### Features

- **Rounded pill shape** - Full rounded borders for modern look
- **Icon support** - Automatic icon sizing with `[&>svg]:size-3`
- **Link badges** - Hover states when used as link (via `asChild`)
- **Focus states** - Ring indicator on keyboard focus
- **Accessibility** - Supports `aria-invalid` for validation states
- **Compact** - Small text size (text-xs) and minimal padding

## ExternalLink

A specialized link component for external URLs that automatically opens in a new tab with proper security attributes.

### Usage

<CodeGroup>
```tsx Basic external link
import { ExternalLink } from "@/components/ui/external-link";

export default function Page() {
  return (
    <p>
      Check out my{" "}
      <ExternalLink 
        href="https://github.com/ahm3tj4f"
        ariaLabel="View GitHub profile"
      >
        GitHub profile
      </ExternalLink>
    </p>
  );
}
```

```tsx With custom className
import { ExternalLink } from "@/components/ui/external-link";

export default function Page() {
  return (
    <ExternalLink 
      href="https://example.com"
      ariaLabel="Visit Example website"
      className="text-blue-600 hover:text-blue-800"
    >
      Visit Example
    </ExternalLink>
  );
}
```

```tsx Email link
import { ExternalLink } from "@/components/ui/external-link";

export default function Page() {
  return (
    <ExternalLink 
      href="mailto:ahmadjafarov02@gmail.com"
      ariaLabel="Send email"
    >
      ahmadjafarov02@gmail.com
    </ExternalLink>
  );
}
```

</CodeGroup>

### Props

<ParamField path="href" type="string" required>
  The URL to link to. Can be:
  - External URL (https://...)
  - Email link (mailto:...)
  - Relative path (/resume.pdf)
</ParamField>

<ParamField path="ariaLabel" type="string" required>
  Descriptive label for screen readers. The component automatically appends "(opens in new tab)" to this label.
  
  Examples:
  - "View LinkedIn profile"
  - "Send email"
  - "Visit company website"
</ParamField>

<ParamField path="children" type="ReactNode" required>
  The visible link text or elements
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes to apply to the link
</ParamField>

### Features

- **New tab** - Automatically opens external links in new tab (`target="_blank"`)
- **Security** - Includes `rel="noopener noreferrer"` for security
- **Screen reader support** - Adds hidden text "(opens in new tab)"
- **Brand styling** - Uses `--brand-orange` CSS variable for color
- **Hover effect** - Underline on hover with offset
- **Next.js Link** - Uses Next.js Link component for routing

### Implementation

```tsx
export function ExternalLink({
  href,
  children,
  className,
  ariaLabel,
}: {
  href: string;
  children: React.ReactNode;
  className?: string;
  ariaLabel?: string;
}) {
  return (
    <Link
      href={href}
      target="_blank"
      rel="noopener noreferrer"
      aria-label={ariaLabel ? `${ariaLabel} (opens in new tab)` : undefined}
      className={`font-semibold hover:underline underline-offset-4 transition-opacity inline-flex items-center gap-1 ${
        className || ""
      }`}
      style={{ color: "var(--brand-orange)" }}
    >
      {children}
      {ariaLabel && <span className="sr-only">(opens in new tab)</span>}
    </Link>
  );
}
```

### Accessibility

1. **aria-label** - Provides descriptive label with "(opens in new tab)"
2. **Screen reader text** - Hidden span announces new tab behavior
3. **Security attributes** - `noopener noreferrer` prevents tab nabbing
4. **Semantic HTML** - Uses proper `<a>` element via Next.js Link

## Common patterns

These UI components are designed to work together:

```tsx
import { Button } from "@/components/ui/button";
import { Badge } from "@/components/ui/badge";
import { ExternalLink } from "@/components/ui/external-link";

export default function ProjectCard() {
  return (
    <article className="border rounded-lg p-6">
      <div className="flex items-center gap-2 mb-4">
        <h3 className="text-xl font-bold">Project Name</h3>
        <Badge>New</Badge>
      </div>
      
      <p className="text-muted-foreground mb-4">
        Check out the{" "}
        <ExternalLink 
          href="https://github.com/user/repo"
          ariaLabel="View project on GitHub"
        >
          source code
        </ExternalLink>
      </p>
      
      <div className="flex gap-2">
        <Button asChild>
          <ExternalLink 
            href="https://demo.example.com"
            ariaLabel="View live demo"
          >
            Live Demo
          </ExternalLink>
        </Button>
        <Button variant="outline">Learn More</Button>
      </div>
    </article>
  );
}
```

## Shared utilities

All three components use common utilities:

- **cn()** - From `@/lib/utils` for conditional class names
- **Radix UI Slot** - For polymorphic component behavior (`asChild` prop)
- **class-variance-authority** - For type-safe variant management
- **TailwindCSS** - For styling with theme support
