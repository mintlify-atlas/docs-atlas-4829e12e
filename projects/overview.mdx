## Technology stack

These projects leverage a wide range of modern technologies:

- **Frontend**: React Native (Expo), Next.js 15, TypeScript, TanStack React Query, shadcn/ui
- **Backend**: FastAPI, Node.js, Server Actions
- **Databases**: PostgreSQL, Redis
- **Tools**: Docker, Zod, React Hook Form, esbuild, WebAssembly
- **Mobile**: Expo Modules, Firebase Cloud Messaging
- **Authentication**: JWT, X.509 certificates, RSA-SHA256 signature verification
