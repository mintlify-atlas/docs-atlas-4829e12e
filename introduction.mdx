## What's included

You get a complete portfolio website with:

- **About section** with personal introduction and professional experience
- **Project showcase** to highlight your best work with descriptions and links
- **Blog functionality** powered by MDX for technical writing
- **Resume page** with downloadable PDF generation
- **Dark mode support** with theme persistence
- **3D animations** using React Three Fiber for visual flair
- **Analytics integration** with Vercel Analytics out of the box

<Info>
  This template uses the App Router architecture from Next.js, providing improved performance, better developer experience, and modern React features like Server Components.
</Info>

## Tech stack highlights

The portfolio leverages cutting-edge technologies:

- **Next.js 16.1.0** - React framework with server-side rendering
- **React 19.2.3** - Latest React with improved hooks and performance
- **TypeScript** - Type safety throughout the codebase
- **TailwindCSS 4** - Utility-first CSS framework
- **shadcn/ui** - High-quality, accessible UI components
- **MDX** - Write blog posts with React components
- **React Three Fiber** - 3D graphics and animations
- **Vercel Analytics** - Privacy-friendly analytics

<Note>
  The project uses pnpm as its package manager, but you can use npm, yarn, or bun if you prefer.
</Note>
