The `Footer` component displays contact information including email, social media links, and resume download. It uses semantic HTML elements for proper structure and accessibility.

## Features

- Semantic HTML with `footer`, `address`, and definition list (`dl`, `dt`, `dd`)
- External links with proper security attributes
- Responsive text sizing
- Monospace font for contact information
- Brand orange color for links

## Structure

The footer uses semantic HTML for better accessibility:

<ParamField path="footer" type="element">
  Main container with `mt-auto` and `pt-8` spacing
</ParamField>

<ParamField path="address" type="element">
  Wraps contact information with `not-italic` to override default styling
</ParamField>

<ParamField path="dl, dt, dd" type="elements">
  Definition list structure where:
  - `dt` - Contact type label (Email, LinkedIn, etc.)
  - `dd` - Contact information with link
</ParamField>

## ExternalLink integration

Each contact link uses the `ExternalLink` component:

<ParamField path="href" type="string" required>
  URL for the contact method (mailto:, https://, or relative path)
</ParamField>

<ParamField path="ariaLabel" type="string" required>
  Descriptive label for screen readers
  
  Examples:
  - "Send email"
  - "View LinkedIn profile"
  - "View GitHub profile"
  - "View resume PDF"
</ParamField>

<ParamField path="children" type="ReactNode" required>
  Display text for the link (usually the contact information)
</ParamField>

## Accessibility

The footer implements accessibility best practices:

1. **Semantic elements** - Uses `footer`, `address`, `dl`, `dt`, `dd`
2. **Not italic address** - Removes default italic styling for better readability
3. **Descriptive labels** - Each link has a clear `ariaLabel`
4. **Screen reader support** - ExternalLink adds "(opens in new tab)" for external links
5. **Margin reset** - `m-0` on `dd` elements prevents unexpected spacing

## Customization

To modify contact links:

1. Update the `CONTACT_LINKS` constant in `src/components/footer.tsx:3-13`
2. Add or remove items in the `<dl>` structure
3. Ensure each item follows the same pattern:
   ```tsx
   <div className="flex items-center gap-3">
     <dt className="w-20 shrink-0 text-sm">Label</dt>
     <dd className="text-xs md:text-sm text-muted-foreground font-mono m-0">
       <ExternalLink href="..." ariaLabel="...">
         Display Text
       </ExternalLink>
     </dd>
   </div>
   ```
